# 2048 AI Demo

🎮 **[點此遊玩 Demo](https://raychen1003.github.io/2048-ai-demo/)**

使用 Expectimax 演算法搭配動態深度調整的 2048 遊戲 AI。

## 功能

- 🧠 多種 AI 策略可選擇
- 📊 即時顯示計算耗時與搜尋深度
- 🎯 動態深度調整機制

## 策略說明

| 策略 | 說明 |
|------|------|
| 動態深度 Expectimax | 根據空格數自動調整深度 (3-4層) |
| 固定深度 3 | 快速但準確度較低 |
| 固定深度 4 | 較慢但準確度較高 |
| 消融實驗 | 驗證各評估因子的貢獻 |

## 演算法

### 評估函數
```
U(B) = Σ(v²) + 100000 × 空格數 + 平滑度³
```

### 動態深度
- 空格 ≥ 6 → 深度 3
- 空格 < 6 → 深度 4
